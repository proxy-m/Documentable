(Documentable::DocPage) method render

Defined as

method render(| --> Hash) { ... }

It's an abstract method. It needs to returns a Hash object following this format:

%(
    document => pod to be rendered as HTML,
    url      => URL assigned to this document
)

Returns the pod is necessary because we need to wrap it in a common HTML format (using Documentable::To::HTML::Wrapper).
