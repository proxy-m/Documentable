(Command line interface) --conf=<Str>

Path to the configuration file (see configuration-file).

If a file called "config.json" is not found at the same directory where Documentable was called, the default config file will be used instead.

Default: "config.json".

This is the structure of this file:

The title-page will be the main headline of that page; pod-root-path will point to the default URL, and will be used mainly for substitution within pages; kinds are menus that will be inserted as part of the header, and every kind will be a sub-menu. The categories section is used to generate different categories in the per-kind page. The sort key tells Documentable to sort that specific page.
