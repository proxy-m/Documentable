(Documentable::Search) method generate-entries

Defined as

method generate-entries(--> Array[Str])

Generates all search entries and store them in an Array. At this moment, it generates three kind of entries:

  • Primary entries: this entries comes from Documentable::Primary objects. We initialize one entry per instance of that class found in the registry as follows:

    category => subkind attribute
    value    => name attribute
    url      => url attribute

  • Secondary entries: this entries comes from Documentable::Secondary objects. Before create any entry of this type, we group all Documentable::Secondary objects by name and after that, we generate one entry per group, as follows:

    category => kind attribute if more than one document is found. Subkind attribute otherwise
    value    => name attribute
    url      => url attribute

  • Reference entries: this entries comes from Documentable::Index objects. One entry per found object.

    category => kind attribute (it's always Kind::Reference)
    value    => name attribute
    url      => url attribute
