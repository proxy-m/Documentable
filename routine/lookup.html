(Documentable::Registry) method lookup

Defined as

method lookup(Str $what, Str $by --> Hash)

This method will classify all Documentable objects in the Documentable::Registry using the attribute $by and will return all of those whose attribute $by is set to $what.

This queris are cached the first time they are made, so next calls to this method will be faster.

Example:

my $registry = (... initialize ...)
 
# query all Documentable objects with $.kind 
# set to Kind::Type 
$registry.lookup(Kind::Type.Str, :by<kind>)
